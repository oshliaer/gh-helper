# GH Helper - Инструкция для AI

## Назначение

GH Helper — инструмент командной строки для работы с комментариями и ревью в GitHub Pull Requests. Позволяет просматривать, фильтровать, отвечать и разрешать комментарии.

## Параметры

- `--code-review-only`, `-cr` : только код-ревью (комментарии к строкам кода)
- `--reviews-only`, `-r` : только ревью
- `--pr-comments-only`, `-pc` : только комментарии к PR (не из ревью)
- `--unresolved-only`, `-u` : только неразрешённые комментарии
- `--count <N>` : ограничить вывод N комментариями (работает с фильтрами)
- `--reply <COMMENT_ID>` : ответить на комментарий (обязательно используется с -m)
- `-m <MESSAGE>` : текст ответа (обязательно при использовании --reply)
- `--resolve` : разрешить комментарий после ответа (работает только с --reply)

**Важно:** Опции фильтрации (`--code-review-only`, `--reviews-only`, `--pr-comments-only`, `--unresolved-only`) являются взаимоисключающими - можно использовать только одну из них за раз.

## Форматы данных

- `PR_NUMBER` : положительное целое число (номер Pull Request'а)
- `COMMENT_ID` : строковый идентификатор комментария (обычно начинается с "PR_" для комментариев к PR или содержит уникальный ID)
- `N` : положительное целое число
- `MESSAGE` : строка текста ответа

## Примеры

- `$> gh-helper 123 -u --count 1` : показать 1 неразрешённый комментарий
- `$> gh-helper 123 -cr --count 3` : показать 3 код-ревью
- `$> gh-helper 123 --reply PR_kwDO... --resolve -m "Fixed as suggested"` : ответить на комментарий и разрешить его
- `$> gh-helper 123 --reply abc123 -m "Addressed your comment"` : ответить на комментарий без разрешения

## Как получить COMMENT_ID

Чтобы ответить на комментарий, нужно сначала получить его ID. Для этого:
1. Выполните команду `gh-helper <PR_NUMBER>` без фильтров или с подходящим фильтром
2. Найдите в выводе интересующий вас комментарий
3. Скопируйте его ID из поля "Comment ID: ..."

## Пошаговый цикл ревью

Предназначен для пошагового разрешения комментариев в PR.

1. Получить следующий неразрешённый комментарий: `$> gh-helper 123 -u --count 1`
2. Используя все доступные инструменты, разрешить комментарий, внеся изменения в код, если требуется
3. Ответить на комментарий и разрешить его: `$> gh-helper 123 --reply COMMENT_ID --resolve -m "<Сообщение>"`
4. Перейти к п.1
